# Here are all the first-class tools, grouped by category:

File System

- view — read files/directories
- create — create new files
- edit — modify existing files
- glob — find files by pattern
- grep — search file contents (ripgrep)

Shell

- powershell — run commands
- write_powershell — send input to a shell session
- read_powershell — read output from a shell session
- stop_powershell — terminate a shell session
- list_powershell — list active shell sessions

Web

- web_fetch — fetch a URL

Agents & Tasks

- skill — invoke a skill
- task — launch a sub-agent
- read_agent — get sub-agent results
- list_agents — list active/completed agents

Memory & Data

- store_memory — persist a fact across sessions
- sql — query the session SQLite database

IDE Integration

- ide-get_diagnostics — get language diagnostics (errors/warnings)
- ide-get_selection — get current editor selection

UX / Meta

- ask_user — ask the user a question
- report_intent — update the displayed intent
- fetch_copilot_cli_documentation — fetch CLI docs
